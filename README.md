## vue介绍

### 目录结构
![vusjs工程目录](./static/images/vuejs工程目录.jpg)

    package.json - 依赖信息
    config - 项目初始化配置
    build - webpack初始化配置
    index.html - 预设首页
    src目录 - 工程目录

入口类： build/webpack.base.conf.js类

    entry: {
      app: './src/main.js'
    }
